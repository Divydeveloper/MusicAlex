# MusicAlex 🎵

A professional Android music player app built using Java and Android Studio.

## Features
- Online playlist
- Play/Pause songs
- Next/Previous controls
- Search songs
- Circular rotating album art
- Modern dark UI
- Animated buttons

## Tech Stack
- Java
- Android Studio
- MediaPlayer API
